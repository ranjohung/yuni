## 3. 会员权益详解

### 3.1 体验版

**适用人群**：初次使用用户

**核心权益**：
- 基础文字对话（每日10次）
- 5个入门场景训练（每周5次）
- 基础情绪识别
- 情绪日记功能
- 每周报告

**限制**：
- 无语音通话
- 仅2D数字人
- LLM响应较慢

### 3.2 基础会员

**适用人群**：有一定使用需求的用户

**核心权益**：
- 文字对话（每日50次）
- 10个场景训练（每周10次）
- 语音通话（30分钟/天）
- 晚安计划
- 2D数字人升级

**附加权益**：
- 更快的LLM响应
- 更长的上下文

### 3.3 标准会员

**适用人群**：深度使用用户

**核心权益**：
- 无限文字对话
- 15个场景训练（每周15次）
- 语音通话（60分钟/天）
- 3D数字人
- 时空穿梭
- 全部场景解锁

**附加权益**：
- 最高优先级LLM调用
- 更多对话历史存储

### 3.4 尊享会员

**适用人群**：重度用户/付费意愿高的用户

**核心权益**：
- 无限文字对话
- 无限场景训练
- 无限语音通话
- 自定义角色系统
- 3D+真人风格数字人
- 全部功能解锁

**附加权益**：
- 专属客服通道
- 优先体验新功能
- 定制化服务
- LLM Pro模型

## 4. 会员购买流程

### 4.1 购买流程

```
选择会员类型 → 选择时长 → 确认订单 → 支付 → 会员生效
```

### 4.2 升级流程

```
当前会员 → 选择更高等级 → 补差价 → 会员升级
```

### 4.3 续费流程

```
会员到期前7天 → 推送续费提醒 → 确认续费 → 支付 → 会员延期
```

## 5. 会员管理

### 5.1 会员状态

| 状态 | 说明 |
|------|------|
| 正常 | 会员权益正常生效 |
| 过期 | 会员到期，未续费 |
| 冻结 | 会员被冻结（违规） |
| 取消 | 用户主动取消订阅 |

| 升级记录 | 会员升级历史 |
| 退款记录 | 会员退款历史 |

### 5.3 退款政策

| 场景 | 退款规则 |
|------|---------|

| 违规冻结 | 不予退款 |

## 7. 会员数据分析

### 7.1 关键指标

| 指标 | 说明 | 目标值 |
|------|------|--------|
| 转化率 | 免费用户转付费用户 | ≥5% |
| 复购率 | 续费用户占比 | ≥60% |
| 留存率 | 会员用户留存 | ≥80% |
| ARPU | 平均收入/用户 | ≥30元 |
| LTV | 用户生命周期价值 | ≥300元 |

### 7.2 分析维度

| 维度 | 分析内容 |
|------|---------|
| 用户行为 | 会员使用频率、时长 |
| 功能使用 | 各功能使用占比 |
| 场景完成 | 场景训练完成率 |
| 付费意愿 | 升级/续费意愿 |

### 7.3 优化方向

| 方向 | 措施 |
|------|------|
| 提升转化 | 优化购买流程，增加试用 |
| 提升复购 | 优化会员权益，增加价值感知 |
| 提升留存 | 增加会员专属内容和活动 |
| 提升ARPU | 推出更高等级会员，增加增值服务 |
