## 3. 数字人渲染

### 3.1 分级渲染策略

| 会员等级 | 渲染类型 | 实现方式 | 动作数量 |
|---------|---------|---------|---------|
| 免费版 | 2D数字人 | Spine 2D | 8个 |
| 基础会员（周卡） | 2.5D Live2D | Live2D Flutter插件 | 12个动作 |
| 标准会员（月卡） | 3D数字人 | Three.js WebView | 25个动作 |
| 尊享会员（年卡） | 3D+真人风格 | Three.js + 自定义材质 | 25个动作+自定义 |

### 3.2 MVP实现：Spine 2D

```dart
class DigitalHuman extends StatefulWidget {
  final Partner partner;
  
  const DigitalHuman({super.key, required this.partner});
  
  @override
  State<DigitalHuman> createState() => _DigitalHumanState();
}

class _DigitalHumanState extends State<DigitalHuman> {
  late SpineWidget _spineWidget;
  
  @override
  void initState() {
    super.initState();
    _spineWidget = SpineWidget.fromAsset(
      'assets/animations/${widget.partner.avatarStyle}.json',
      onLoaded: (controller) {
        controller.setAnimation(0, 'idle', true);
      },
    );
  }
  
  void playAnimation(String animationName) {
    _spineWidget.controller?.setAnimation(0, animationName, false);
  }
  
  @override
  Widget build(BuildContext context) {
    return _spineWidget;
  }
}
```

## 5. 时空穿梭

### 5.1 功能流程

```
用户点击时空穿梭
    │
    ▼
检查穿梭券数量
    │
    ├── 有券 → 选择历史场景
    │           │
    │           ▼
    │       重新体验该场景
    │           │
    │           ▼
    │       结束后生成学习卡片
    │
    └── 无券 → 提示获取方式
            │
            ▼
        每日登录赠送 / 积分兑换 / 看广告
```

### 5.2 穿梭券获取方式

| 获取方式 | 数量 | 频率 |
|---------|------|------|
| 每日登录赠送 | 1张 | 每日 |
| 积分兑换 | 1张 | 100积分 |
| 看广告 | 1张 | 每日限2次 |
| 会员赠送 | 3-10张 | 周/月/年 |
