# 模拟训练场景设计

> 版本：v1.0  
> 适用对象：内容团队、产品团队  
> 模块：内容运营

---

## 1. 场景设计规范

### 1.1 场景设计原则

| 原则 | 说明 |
|------|------|
| 真实性 | 场景贴近现实生活 |
| 循序渐进 | 难度从低到高逐步提升 |
| 目标明确 | 每个场景有清晰的训练目标 |
| 即时反馈 | 提供及时的评估和建议 |
| 趣味性 | 保持场景的吸引力 |

### 1.2 场景结构定义

```
场景ID → 场景名称 → 所属阶段 → 难度等级 → 预计时长 → 解锁条件
    ↓
场景背景 → 角色设定 → 对话流程 → 评估维度 → 学习卡片
```

### 1.3 难度分级标准

| 等级 | 难度 | 特点 | 示例 |
|------|------|------|------|
| 1 | 入门 | 简单场景，明确指引 | 初次见面打招呼 |
| 2 | 基础 | 中等复杂度，需要思考 | 咖啡厅破冰对话 |
| 3 | 进阶 | 较高复杂度，需要技巧 | 职场沟通 |
| 4 | 高级 | 复杂场景，综合运用 | 冲突解决 |

### 2.2 场景详细设计

#### S001 - 初次见面打招呼

**场景背景**：你在朋友的聚会上遇到一个新朋友。

**角色设定**：
- 对方：友好型，初次见面略显紧张
- 用户目标：自然地打招呼并开启对话

**对话流程**：
1. AI发起："嗨，你好！我是小明，你是？"
2. 用户回应
3. AI回应：根据用户回应进行互动

**评估维度**：
- 话题开启自然度（30%）
- 回应友好度（30%）
- 自信程度（40%）

**学习卡片**：初次见面打招呼技巧

#### S003 - 咖啡厅破冰

**场景背景**：你在咖啡厅等待朋友时，旁边座位的人主动和你搭话。

**角色设定**：
- 对方：开朗型，喜欢咖啡和阅读
- 用户目标：建立轻松的对话氛围

**对话流程**：
1. AI发起："你好，请问你也经常来这家咖啡厅吗？"
2. 用户回应
3. AI回应：根据用户回应展开话题

**评估维度**：
- 话题开启自然度（25%）
- 回应长度（20%）
- 信息分享度（25%）
- 开放性提问（30%）

**学习卡片**：破冰对话技巧

**角色设定**：
- 对方：专业型，注重效率和结果

## 6. 评估标准

### 6.1 评估维度权重

| 维度 | 基础权重 | 适用场景 |
|------|---------|---------|
| 话题开启自然度 | 20% | 所有场景 |
| 回应长度 | 15% | 所有场景 |
| 自信程度 | 20% | 所有场景 |
| 信息完整性 | 15% | 信息分享场景 |
| 开放性 | 10% | 对话场景 |
| 情绪控制 | 20% | 冲突场景 |

### 6.2 评分等级

| 等级 | 分数范围 | 评价 |
|------|---------|------|
| S | 90-100 | 优秀，完美完成 |
| A | 80-89 | 良好，表现出色 |
| B | 70-79 | 中等，基本完成 |
| C | 60-69 | 及格，需要改进 |
| D | <60 | 不及格，重新练习 |

### 6.3 反馈机制

| 分数 | 反馈类型 |
|------|---------|
| ≥80 | 正面鼓励 + 进阶建议 |
| 60-79 | 具体改进建议 |
| <60 | 详细指导 + 重新练习 |

## 8. 场景更新策略

### 8.1 更新频率

| 类型 | 频率 | 说明 |
|------|------|------|
| 新增场景 | 每月1-2个 | 根据用户反馈添加新场景 |
| 优化场景 | 每季度 | 根据数据分析优化现有场景 |
| 节日场景 | 节日前2周 | 添加节日相关场景 |

### 8.2 用户反馈收集

| 反馈渠道 | 方式 | 频率 |
|---------|------|------|
| 场景结束后 | 满意度评分 | 每次完成 |
| 定期调研 | 问卷 | 每月 |
| 社区反馈 | 论坛/社群 | 实时 |

### 8.3 数据指标

| 指标 | 目标值 |
|------|--------|
| 场景完成率 | ≥80% |
| 平均得分 | ≥75分 |
| 满意度评分 | ≥4.5/5 |
| 重复练习率 | ≥30% |
