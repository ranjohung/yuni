## 1. 文档概述

### 1.1 产品定位

「与你」是一款AI驱动的数字人社交模拟训练App，通过高拟真数字人实时互动，结合认知行为疗法（CBT）、依恋理论、非暴力沟通（NVC）等心理学框架，为用户提供零压力的社交场景模拟训练与情感陪伴。

### 1.2 一句话价值主张

> **"先和TA练一遍，再去面对真实世界。"**

### 1.3 核心差异化

| 维度 | 传统AI聊天 | 「与你」 |
|------|-----------|---------|
| 目标 | 陪伴/闲聊 | 社交技能训练 |
| 心理学框架 | 无 | CBT + 依恋理论 + NVC |
| 训练结构 | 无 | 六阶段社交生命周期 |
| 反馈机制 | 无 | 即时反馈 + 评估报告 + 情绪日记 |
| 成长闭环 | 无 | 时空穿梭 + 学习卡片 + 晚安计划 + 每周报告 |
| 形象联动 | 无 | 与悦己颜值社联动 |
| 留存体系 | 无 | 每日签到 + 晚安计划 + 里程碑 + 关系危机 |

### 1.4 技术目标

| 指标 | MVP目标 | 正式版目标 |
|------|---------|-----------|
| 平台支持 | iOS / Android | iOS / Android |
| AI推理响应 | < 3秒（流式输出） | < 2秒 |
| 数字人渲染 | 2D动态立绘 | 3D ≥ 30fps |
| 语音通话延迟 | 1-2秒（MVP可选功能） | < 500ms |
| 并发承载 | 1万 DAU | 10万 DAU |

## 2. 产品定位与用户画像

### 2.1 核心目标用户

18-35岁、有社交焦虑或情感陪伴需求的都市年轻人（女性占比预计65-70%）。

### 2.2 四类用户画像

#### 第一类：社交焦虑人群（核心用户）
| 维度 | 描述 |
|------|------|
| 年龄 | 22-30岁 |
| 特征 | 害怕社交场合、不会聊天、容易冷场、被拒绝后过度自责 |
| 痛点 | 想练习社交但没有安全的环境 |
| 需求 | 零压力的社交训练场 |

#### 第二类：职场新人（高频用户）
| 维度 | 描述 |
|------|------|
| 年龄 | 22-28岁 |
| 特征 | 刚入职、需要面试/汇报/向上沟通 |
| 痛点 | 紧张、表达混乱、不敢开口 |
| 需求 | 职场场景模拟演练 |

#### 第三类：情感陪伴需求者
| 维度 | 描述 |
|------|------|
| 年龄 | 18-35岁 |
| 特征 | 孤独、寻求陪伴、缺乏倾诉对象 |
| 痛点 | 有情绪需要释放、渴望被倾听 |
| 需求 | 随时在线的情绪支持 |

#### 第四类：自我提升爱好者
| 维度 | 描述 |
|------|------|
| 年龄 | 25-35岁 |
| 特征 | 关注个人成长、愿意为自我提升付费 |
| 痛点 | 理论知识多但缺乏练习机会 |
| 需求 | 可量化的社交能力训练+每周报告 |

### 2.3 用户兴趣偏好
- 关注个人成长、心理健康
- 喜欢养成类游戏/互动叙事
- 为情绪价值付费意愿高
- 对AI/科技产品接受度高

## 3. 核心技术架构

### 3.1 技术选型总表

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| 客户端框架 | Flutter | 跨端开发，iOS/Android一套代码 |
| 数字人渲染(MVP) | Spine 2D骨骼动画 | 免费，轻量，快速上线 |
| 数字人渲染(V1.1+) | 魔珐星云SDK / Three.js | 高拟真3D |
| 实时音视频 | 声网Agora RTC | 全双工语音通话 |
| LLM对话引擎(主力) | DeepSeek-Chat | 效果最优，按token付费 |
| LLM对话引擎(兜底) | Ollama + Qwen2.5:14B | 免费，自动降级 |
| TTS语音合成 | Microsoft Azure TTS | 多音色，晚安计划语音 |
| 情绪识别 | DeepSeek文本分析 | MVP以文本为主 |
| 长期记忆 | MySQL JSON字段 | MVP不上向量数据库 |
| 后端框架 | Node.js + Express | 轻量快速迭代 |
| 数据库 | MySQL 9.6 | 业务数据持久化 |
| 缓存 | Redis | Session、热点数据 |
| 对象存储 | 阿里云OSS | 用户照片、素材、晚安语音 |
| 定时任务 | node-cron | 晚安计划推送、朋友圈生成、每周报告 |

### 3.2 LLM混合引擎架构

```
用户输入 → LLM路由器
  ├─ 体验用户(1次) → DeepSeek
  ├─ 免费用户 → Ollama + 10%偶遇DeepSeek
  ├─ 日卡/周卡/月卡 → DeepSeek
  └─ 年卡 → DeepSeek高优先级
       → 健康检查失败 → 自动降级Ollama
       → 输出给用户
```

### 3.3 成本控制策略

| 策略 | 措施 | 节省 |
|------|------|------|
| LLM混合路由 | DeepSeek+Ollama | 减少API约80% |
| 内容预编译 | 预设对话树 | 60%走预设 |
| 本地优先 | 好感度客户端计算 | 服务器成本 |
| 免费素材优先 | Spine 2D+无版权 | SDK授权费 |
| MySQL+JSON | MVP不上向量库 | 服务器成本 |

## 5. 数字人系统

### 5.1 分级渲染

| 会员等级 | 形态 | 渲染方式 |
|---------|------|---------|
| 体验/免费 | 2D动态立绘 | Spine 2D骨骼动画 |
| 周卡 | 2.5D Live2D | 本地渲染 |
| 月卡 | 3D数字人 | Three.js + glTF |
| 年卡 | 3D+真人风格 | Three.js + 自定义 |

### 5.2 MVP方案：Spine 2D

- 免费版制作4个角色立绘动画
- 5种表情：微笑、眨眼、点头、歪头、挥手
- 后期可迁移到Live2D/Three.js

### 5.3 灵魂架构

| 流浪者 | 追逐新鲜感 | 顾星河 |

| 场景 | 阶段 | 难度 | 轮次 | 教学点 |

### 6.3 场景训练流程

```
用户点击[开始训练] → 场景介绍页

```

## 7. 好感度与关系系统

### 7.1 好感度等级

| 等级 | 名称 | 区间 | 解锁内容 |
|------|------|------|---------|
| Lv.0 | 陌生人 | 0 | 基础聊天 |
| Lv.1 | 熟人 | 1-200 | 朋友圈、场景训练 |
| Lv.2 | 朋友 | 201-400 | 自定义称呼、NVC |
| Lv.3 | 好友 | 401-600 | 时空穿梭、学习卡片 |
| Lv.4 | 知己 | 601-1000 | 语音通话、依恋分析 |

### 7.2 增减规则

| 行为 | 变化 |
|------|------|
| 每日首次登录 | +5 |
| 完成训练 | +10~30 |
| 高质量对话(LLM) | +5 |
| NVC沟通 | +15 |
| 冷场 | -3~-8 |
| 3天未登录 | -5/天 |
| 分手 | 归零 |

### 7.3 关系危机系统

- 7天未登录→危机状态→AI主动询问
- 一次高质量互动解除
