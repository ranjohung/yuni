## 一、隐私声明

### 1. 信息收集与使用

**1.1 我们收集的信息类型**

| 信息类别 | 收集方式 | 用途 | 是否必填 |
|---------|---------|------|---------|
| 手机号码 | 用户注册时填写 | 账号注册、登录验证 | 必填 |
| 昵称 | 用户设置 | 显示用户名称 | 可选 |
| 头像照片（如有） | 用户上传 | 生成数字人形象 | 可选 |
| 对话内容 | 用户与AI互动时产生 | 提供对话服务、生成学习卡片 | 必填（功能需求） |
| 使用时长 | 系统自动记录 | 防沉迷提醒、使用统计 | 必填 |
| 实名信息（如有） | 用户实名认证时 | 年龄验证、合规要求 | 条件必填 |

**1.2 信息使用原则**
- 对话内容仅用于提供「与你」服务，绝不用于广告投放
- 不向任何第三方出售用户数据
- 匿名化统计数据可能用于产品优化

### 2. 未成年人保护

**2.1 年龄限制**
- 14岁以下用户仅可使用基础社交场景（3个场景）
- 14-18岁用户每日使用限制30分钟，屏蔽亲密互动功能
- 18岁以上用户开放全部功能

**2.2 实名认证**
- 用户需完成实名认证方可解锁全部功能
- 实名信息仅用于年龄验证，验证后仅保存"已成年/未成年"标记
- 实名信息保存期限：用户注销账号后立即删除

### 3. AI透明声明

- 您与「与你」App中所有数字角色的互动均为AI驱动
- 所有对话界面均显示"我是AI助手"标识
- 数字角色不具备真实人类的情感意识和自主意志
- 请勿将AI建议作为专业心理咨询的替代

### 4. 数据存储与安全

| 数据类别 | 存储位置 | 加密方式 | 保留期限 |
|---------|---------|---------|---------|
| 对话记录 | 服务器加密存储 | TLS传输加密 + AES存储加密 | 用户可随时删除 |
| 个人信息 | 服务器加密存储 | AES-256 | 账号存续期间 |
| 实名信息 | 服务器加密存储 | 哈希处理 | 验证后仅保留标记 |

## 二、用户协议（摘要版）

> 完整版建议请法律顾问起草。以下为功能相关的关键条款。

### 1. 服务说明

「与你」是一款AI驱动的社交模拟训练App，提供：
- AI数字人对话互动
- 社交场景模拟训练
- 社交能力评估与成长追踪
- 心理健康引导（CBT/NVC框架）

### 2. 用户义务

| 义务 | 说明 |
|------|------|
| 提供真实信息 | 注册及实名认证时需提供真实信息 |
| 不使用违法内容 | 不得输入违法、色情、暴力内容 |
| 不满18岁需监护人同意 | 未成年用户使用需监护人知情同意 |
| 不滥用服务 | 不得使用自动化工具、不得恶意攻击系统 |
| 不冒充他人 | 不得冒用他人身份注册 |

### 3. 内容版权

- 用户与AI的对话内容版权归用户所有
- AI即时生成的才艺内容（诗歌/故事等）版权归平台所有
- 用户上传的照片/形象仅用于App内功能

### 4. 免责声明

- AI建议仅供参考，不作为专业心理治疗或医疗建议
- 如用户有严重心理困扰，建议寻求专业心理咨询
- 平台对用户因使用服务产生的现实社交结果不做任何保证

### 5. 服务终止

| 终止原因 | 处理方式 |
|---------|---------|
| 用户主动注销 | 立即删除所有数据 |
| 用户违规使用 | 先行警告，严重者封禁账号 |
| 平台停止运营 | 提前30天通知，提供数据导出通道 |

### 方案选择

| 方案 | 成本 | 说明 | 优先级 |
|------|------|------|--------|
| **方案A：手机号实名（推荐）** | **免费** | 利用运营商三要素（手机号+姓名+身份证号）验证 | ⭐ 首选 |
| 方案B：阿里云实人认证 | ¥0.18/次 | 标准人脸识别实名，仅用于照片上传等高级功能 | 备用 |

### 实施步骤

```
用户注册（手机号+验证码）
    │
    ├── 方案A（免费）：
    │    ├─ 用户填写姓名+身份证号
    │    ├─ App调用运营商三要素验证API
    │    └─ 验证通过 → 标记"已实名"
    │
    └── 方案B（¥0.18/次，仅高级功能需要时触发）：
         ├─ 用户点击"上传照片生成数字人"或"解锁高级内容"
         ├─ 跳转阿里云实人认证页面（人脸识别）
         ├─ 验证通过 → 解锁对应功能
         └─ 日成本约¥1.8（日均10新用户）
```
